# Resume Builder

A full-stack AI-powered resume builder that helps users create, edit, and export professional resumes with authentication, dashboard management, and modern templates.

## Live Demo
https://ai-resume-builder-g43e.vercel.app/

## Features
- User authentication (sign up, sign in, forgot/reset password)
- Responsive marketing and dashboard UI
- Resume CRUD operations with structured editor sections
- AI tools for summary and bullet improvement
- Resume template preview and PDF export

## Tech Stack
- Next.js (App Router)
- React + TypeScript
- Tailwind CSS
- Prisma ORM
- PostgreSQL
- Stripe
- OpenAI API

## How to Run

```bash
cd resume-builder
npm install
npm run dev
```
